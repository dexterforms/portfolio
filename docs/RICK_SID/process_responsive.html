RESEARCH AND DISCOVERY

  • Assessed and operationalize your goals.

  • Analyze competitor trends, strengths and weaknesses.

  • Pinpoint objectives of the site and motivations of average visitor (for example, a banking web site would want to allow users to quickly access their funds and banking info while a restaurant would want its menu and hours easily found).

  • Lay out a detailed timeline for the life of the project.

SKETCH AND IDEA GENERATION

  • Colors, schemes, graphics and logos are tried and tested.

  • Written web content is created and placed.

  • Navigation is laid out and a rough site map is constructed.

CODE DEVELOPMENT

  • Custom code is written and the programming begins.

  • Social media accounts are created and embedded into site.

BETA TESTING

  • Every feature is tested and perfected; bugs are worked out.

LAUNCH

  • Rejoice! Yoursite springs to life!.

POST LAUNCH

  • React to changing conditions, analytics or needs by updating functions.

  • Monitor success of site and ameliorate any shortcomings.

  • Expand or rewrite code as needed to accommodate new developments.
